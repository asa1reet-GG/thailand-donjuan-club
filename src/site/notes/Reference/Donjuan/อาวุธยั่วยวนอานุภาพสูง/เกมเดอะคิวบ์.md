การตีความ THE CUBE

หมายเหตุ: ถามคำถามตามลำดับที่ถูกต้องนะครับ (ลูกบาศก์, บันได, ดอกไม้, ม้า, พายุ - ต้องตามลำดับนี้) และตีความหลังจากเธอจินตนาการครบทั้ง 5 อย่างแล้วเท่านั้น เผลอๆ คุณอาจจะอุบไต๋บางส่วนไว้เพื่อหาเรื่องคุยต่อคราวหน้าก็ได้ (แบบว่า "ผมต้องขอคิดเรื่องลักษณะม้าของคุณอีกหน่อย แล้วจะบอกนะถ้าตีความได้เข้าท่าแล้ว") บางคนในบอร์ด ASF อาจจะไม่เห็นด้วยกับการตีความบางอย่าง แต่ไม่เป็นไรครับ ผมคิดว่าถ้าการตีความมันฟังดูสมเหตุสมผลสำหรับผม มันก็น่าจะถูกแหละ

การตีความแต่ละชิ้นมี 2 ส่วนประกอบ คือตัววัตถุเอง และความสัมพันธ์ของมันกับวัตถุอื่นรวมถึงภาพรวมทั้งหมด ขนาดและตำแหน่งของสิ่งต่างๆ เป็นเรื่องสัมพัทธ์ เปรียบเทียบกันเองและเปรียบเทียบกับฉากทะเลทราย ยิ่งกว่านั้น การตีความไม่ได้มีกฎตายตัว มันเป็นแค่วิธีทำความเข้าใจภาพในหัวของผู้หญิง แต่ตรงนี้แหละคือส่วนที่ดีที่สุด เพราะสำหรับพวกเราใน ASF ความยืดหยุ่นนี้ช่วยให้เราตีความในแบบที่ช่วยเรื่องการจีบสาวได้ (การตีความเชิงลบถือเป็นการแซวแบบ Neg-hit! คุณสามารถใส่คำทำนายเชิงลบเล็กๆ น้อยๆ ได้เมื่อจำเป็น โดยที่ภาพรวมยังดูสมเหตุสมผลอยู่)

1. ลูกบาศก์ (THE CUBE):

แทนความคิดที่ผู้หญิงมีต่อตัวเอง ลูกบาศก์ใหญ่ยักษ์กินพื้นที่เกือบทั้งฉาก (Field of View หรือ FOV) แปลว่าเธออีโก้สูง หลงตัวเองมาก ลักษณะอื่นๆ ของลูกบาศก์อาจหมายถึง:

ลูกบาศก์จิ๋ว => รู้สึกตัวเล็ก ไม่สำคัญ ถูกมองข้าม ถ่อมตัว
ลูกบาศก์วางอยู่บนพื้น => โดยทั่วไปคือมีจุดยืนที่มั่นคงในโลกความเป็นจริง
ลูกบาศก์อยู่ไกลลิบ => รู้สึกถูกทิ้งไว้ข้างหลังในชีวิต
ลูกบาศก์ลอยอยู่บนฟ้า => ช่างฝัน จินตนาการสูงแต่ไม่มองความจริง
ลูกบาศก์อยู่เหนือเส้นขอบฟ้าบางส่วน => มีความทะเยอทะยาน
ลูกบาศก์อยู่ใต้เส้นขอบฟ้า => ไม่ค่อยทะเยอทะยาน
ลูกบาศก์วางตะแคงบนสัน => ชีวิตอาจจะไม่มั่นคงหรือเปล่า?
ลูกบาศก์ทำจากวัสดุแข็งทึบ => เห็นคุณค่าในตัวเอง บุคลิกหนักแน่น
ลูกบาศก์ทำจากทอง => คิดว่าตัวเองเลอค่ามาก
ลูกบาศก์ทำจากแก้วหรือใส => มองว่าตัวเองบริสุทธิ์
ลูกบาศก์เต็มไปด้วยเมือก => เกลียดตัวเองสุดๆ
ลูกบาศก์ข้างในกลวง => รู้สึกว่างเปล่า ไม่ได้รับการเติมเต็มอย่างรุนแรง
[ตัวอย่างที่น่าสนใจ: มีผู้หญิงคนหนึ่งที่ผมรู้จักดี จินตนาการเป็นรูบิคที่ถูกเด็กบิดเล่น ผมไม่แปลกใจเลยเพราะเธอมีปมเรื่องถูกกลั่นแกล้งรุนแรงและหวาดระแวงไปหมด มองว่าคนอื่นโง่และทำตัวเป็นเด็ก (พวกชอบทำตัวเหนือกว่าคนอื่น) รู้สึกเหมือนถูกโลกทำร้าย และอารมณ์แปรปรวนสุดๆ]

2. บันได (THE LADDER):

แทนโครงสร้างทางสังคมที่ใกล้ชิดของเธอ (เพื่อน; และครอบครัวในบางกรณี)
บันไดยาว มีหลายขั้น => สังคมกว้าง เพื่อนเยอะ ชอบเข้าสังคม
บันไดทำจากวัสดุแปลกๆ => รู้สึกว่าเพื่อนตัวเองเพี้ยน ต่างจากคนปกติมาก
บันไดมีขั้นน้อย => มีเพื่อนสนิทน้อย
บันไดสภาพไม่ค่อยดี => เชื่อว่าคนรอบข้างมีปัญหาชีวิต
บันไดอยู่ห่างจากลูกบาศก์ => ไม่ยอมให้ใครเข้าใกล้ตัวเองมากเกินไป ถือตัว มีกำแพงกั้น
บันไดพาดอยู่กับลูกบาศก์ => รู้สึกว่าตัวเองทำให้เพื่อนเยอะ คอยซัพพอร์ตเพื่อนมากกว่าเพื่อนซัพพอร์ตตัวเอง รู้สึกว่ามีคนมาเกาะแกะพึ่งพา
บันไดอยู่บนลูกบาศก์ => รู้สึกว่าเพื่อนหรือครอบครัวชอบบงการ รู้สึกอึดอัด
บันไดใหญ่กว่าลูกบาศก์มาก => รู้สึกตัวเล็กในกลุ่มสังคม
บันไดรองรับลูกบาศก์ (เช่น บันไดอยู่ข้างล่างลูกบาศก์) => รู้สึกว่าคนใกล้ชิดคอยสนับสนุนความสำเร็จของเธอ
บันไดแข็งแรง => รายล้อมด้วยคนที่เข้มแข็ง รู้สึกปลอดภัย
บันไดถูกเผา => รู้สึกว่ารายล้อมไปด้วยคนที่ชีวิตพังพินาศและกำลังทำลายตัวเอง

3. ดอกไม้ (THE FLOWERS):

แทนเรื่องลูกหรือเด็กๆ ในชีวิตของเธอ

จำนวนดอกไม้ => จำนวนลูกที่มีหรืออยากมี หรืออยากให้มีอยู่รอบๆ (ดู * ข้างล่าง)
ดอกไม้อยู่ใกล้ลูกบาศก์ => รู้สึกผูกพันกับลูกที่มีหรือจะมีมาก
ดอกไม้อยู่ไกล => ไม่อยากมีลูก
ดอกไม้บานสะพรั่ง => รู้สึกดีกับชีวิตของลูกๆ
ดอกไม้เละเทะ => รายล้อมไปด้วยเด็กที่มีปัญหา
* ดอกไม้เยอะแยะเต็มไปหมด => น่าจะทำงานเกี่ยวกับเด็ก หรืออยากทำ (สาวคนหนึ่งที่ผมรู้จักตอบแบบนี้ เธอเป็นครูประถม)
ดอกไม้สั่นไหวในสายลม => รู้สึกว่าเด็กๆ ในชีวิตกำลังลำบาก
ดอกไม้อยู่รอบๆ หรือคลุมลูกบาศก์ => รู้สึกรับมือกับเด็กๆ ไม่ไหว
ดอกไม้ถูกบันไดกั้นออกจากลูกบาศก์ => รู้สึกว่าเพื่อนหรือครอบครัว (เข้ามายุ่ง/จะเข้ามายุ่ง/อาจจะเข้ามายุ่ง) เรื่องความสัมพันธ์ระหว่างเธอกับลูก
ดอกไม้สวยงาม (กุหลาบ ป๊อปปี้ ฯลฯ) => มองว่าเด็กๆ สวยงามน่ารัก

4. ม้า (THE HORSE):

แทนความคิดที่เธอมีต่อคนรัก (หรือคนรักที่เธอคิดว่าอยากได้หรือจะมี)

ม้าตัวใหญ่ แข็งแรง => อยากได้ผู้ชายที่ปกป้องได้ เข้มแข็ง
สีของม้า => อาจหมายถึงเชื้อชาติของแฟนที่อยากได้ (ครูสาวคนนั้นชอบหนุ่มละติน ม้าของเธอก็สีน้ำตาล)
ม้าอยู่ใกล้ลูกบาศก์ => อยากให้คนรักใกล้ชิดทั้งกายและใจ
ม้าอยู่ห่างออกไป => ยังสงวนท่าที ไม่เปิดใจให้คนรักเต็มที่
ม้าเลียหรือดมลูกบาศก์ => จินตนาการหรือหวังว่าจะได้รับการเอาอกเอาใจ
ม้าตัวเล็ก เชื่อง => อยากได้แฟนที่ข่มได้
ม้าป่าพยศ => อยากได้ผู้ชายดิบๆ ที่ไม่เชื่องและไม่มีวันเชื่อง
ม้าถูกล่ามไว้ => อยากควบคุมหรือจำกัดอิสระเขามากๆ
ม้ากระทืบลูกบาศก์ => เคยถูกคนรักทำร้ายอย่างรุนแรงหรือรู้สึกแบบนั้น
ม้าทำลายดอกไม้ => รู้สึกว่าคนรักจะไม่ดีกับลูกๆ ของเธอ (แม่เลี้ยงเดี่ยวมักจะเป็นแบบนี้บ่อย)
ม้ายุ่งกับบันได => มีความขัดแย้งระหว่างแฟนกับเพื่อนของเธอ
ม้าอยู่ไกลหรือเดินหนี => รู้สึกถูกทอดทิ้ง
ม้าถูกลูกบาศก์กั้นออกจากดอกไม้ => รู้สึกว่าตัวเองต้องดูแลลูกและคอยจัดการความสัมพันธ์ระหว่างแฟนกับลูกเอง
ม้าอ่อนแอ => มองภาพว่าจะต้อง (ติดแหง็ก?) อยู่กับผู้ชายขี้แพ้
[ตัวอย่าง: สาวคนหนึ่งที่ผมรู้จักเห็นม้าวิ่งวนอยู่ในสนามที่ถูกล้อมรั้ว เธอเป็นประเภทชอบบงการ ชอบกำหนดขอบเขต และชอบดูปฏิกิริยาของผู้ชายเวลาเธอทดลองปั่นหัวเล่นกับอารมณ์หรือพฤติกรรมของเขา]

5. พายุ (THE STORM):

แทนความคิดเรื่องปัญหาและอุปสรรคในชีวิต

พายุอยู่ไกลๆ => ตอนนี้ปัญหาชีวิตยังไม่รุมเร้า
พายุกำลังมา => กลัววิกฤตในอนาคต
พายุกำลังจากไป => เพิ่งเจอปัญหามาแต่รู้สึกว่ามันจบแล้ว
พายุใหญ่ยักษ์ ครอบคลุมพื้นที่ => รู้สึกว่าชีวิตกำลังวิกฤตหนัก
พายุอยู่ไกลๆ ผ่านไปโดยไม่กระทบ 4 อย่างนั้นเลย => รู้สึกว่าชีวิตค่อนข้างไร้ปัญหา มีเรื่องปวดหัวน้อย
พายุลูกเล็ก => รู้สึกมั่นใจว่าจะรับมือกับปัญหาที่จะเข้ามาได้
